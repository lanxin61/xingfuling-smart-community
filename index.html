Project Overview

项目概览

本项目以金象社区为核心样板试点，整合社区场地、街道政务、职业院校、 社会服务与智慧科技资源，建设集生活照料、精神文娱、康养服务、人才培育、 智慧赋能、就业赋能于一体的智慧银发融合社区养老服务平台。

01

建设目标

  • 补齐社区养老服务短板，完善精神文化、康养社交、智慧健康管理等服务体系。
  • 培育社区主理人、家政服务、康养服务与机构导流团队，形成自主运营能力。
  • 深化校企融合，建设养老实训与科研创新阵地，提升服务专业化水平。
  • 盘活社区场地与人力资源，孵化社区家政与养老服务岗位。
  • 探索可复制、可盈利、可持续的社区养老运营模式。
02

服务对象

  • 社区中老年群体：自理、半失能、高龄、慢病、独居空巢老人。
  • 老人家庭与照护者：子女、家属及长期照护者。
  • 社区普通居民：有家政、助餐、健康、文化活动需求的居民。
  • 失业及灵活就业人员：希望参与养老、家政、社区运营等岗位的人群。
  • 合作机构与服务单位：养老机构、职业院校、医疗健康与适老化服务商。

Service System

九大服务体系

以社区为服务入口，链接家庭、机构、院校与专业服务资源， 为长者和家庭提供从日常照护到康养支持的一站式服务。

Operating Model

运营特色

社区可信入口

依托社区属地信任优势，将养老服务、家政便民、机构甄选和健康管理前置到居民身边。

专业服务协同

整合医护、康复、护理、旅居、适老化改造、职业培训等资源，形成标准化服务供给。

可持续运营

兼顾公益属性与市场化能力，通过人才培育、家政服务、机构导流和增值服务形成运营闭环。
